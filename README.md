# MVC Product Shop

## 1. Install dependencies

```bash
npm install
```

## 2. Create DB and tables

Run SQL from `sql/init.sql` in your MySQL client.

## 3. Configure environment variables

Set values in `.env`:

- `DB_HOST`
- `DB_USER`
- `DB_PASSWORD`
- `DB_NAME`
- `PORT`

## 4. Start app

```bash
npm run dev
```

or

```bash
npm start
```

App URL: <http://localhost:3000>
